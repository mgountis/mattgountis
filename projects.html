NBA Data Science Project

Building an NBA Champion

In my final semester of school I completed a data science internship with a professor from the university. Throughout the semester I completed several Coursera courses to learn Python (specifically Pandas, MatPlotLib, and Numpy). Using available NBA statistics for both individual players and teams I completed a data analysis to recommend a team of players that would be capable of winning the NBA Championship. At the link is the completed Jupyter Notebook (using Kaggle) of my final presentation including several visualizations of key data.




SharePoint Field Updater


While working to transfer large data sets from IBM Lotus Notes to MS SharePoint, I created a custom application to automate the process. Using C# and Windows WPF, the application accepted CSVs of large data sets and programatically created SharePoint Sites and lists, then populated those lists with the supplied data. This made the process of data migration much more efficient, allowing the team to complete more sites more quickly than we had originally hoped.

SQL Fundraising WebApp


As the semester project for my senior databases course, I created a custom Web Application (Using HTML, CSS, and PHP) to interface with databases running in the backend. In building this application it required the building the web front end as well as using Microsoft SQL Server to build the backend. The database kept track of customers, delivery addresses, inventory, and orders. Through the web interface you could submit orders and insert them into tables and managers could sign in and view order histories and delete users among other things.

Color Picking Game Web App

Color Picker

This is a simple web application utilizing javacript for the logic and some light styling. It generates up to six random colors on the RGB spectrum, chooses one of them randomly as the "correct" answer, and then is a game where the user has to select colors until the right one is chosen. There is functionality to "play again" without refreshing and two difficulties.
